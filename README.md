# インターネットTVサービス

mysqlで実装したインターネットTVサービスのデータベースです。

---

## プロジェクト概要

| 項目 | 内容 |
|---|---|
| OS | macOS |
| データベース | MySQL |
| MySQLバージョン | 8.0以上 |

### 主な機能
- チャンネル管理
- 番組・シーズン・エピソード管理
- ジャンル管理
- 番組枠管理
- 視聴数の記録・集計

## セットアップ手順

### 1. MySQLのインストール

```bash
brew install mysql
```

### 2. MySQLの起動

```bash
brew services start mysql
```

### 3. MySQLへのログイン

```bash
mysql -u root -p
```

### 1. データベースの作成

```zsh
source /path/to/ddl.sql
```

### 2. テーブルの作成

```zsh
source /path/to/dml.sql
```

### 3. サンプルデータの投入

```zsh
source /path/to/sample.sql
```
